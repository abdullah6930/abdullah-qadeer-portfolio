Mesh Generator - Unity Plugin

Mesh Generator is an open-source Unity plugin designed for real-time manipulation of mesh vertices during play mode. This powerful tool empowers developers to create and modify 3D shapes and environments directly within their games, enhancing both flexibility and creative possibilities.

Key Features

  • Dynamic Mesh Generation: Create meshes from predefined shapes such as Quad, Box, and Pyramid with ease.
  • Real-Time Vertex Editing: Change vertex positions in play mode using mouse or touch input, similar to manipulating GameObjects in the Unity Editor.
  • Adjustable Parameters: Modify mesh width, height, and volume directly from the user interface, providing control over mesh dimensions.
  • Custom Mesh Support: Easily manipulate vertex positions of custom meshes, offering flexibility for diverse project needs.

Current Mesh Types

  • Quad
  • Box
  • Pyramid

Future Developments

  • Terrain Generation
  • Local Shape Saving Capability
  • Building Generation

Technologies Used: Unity, C#
