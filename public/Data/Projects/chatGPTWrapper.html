ChatGPT Wrapper for Unity

A simple ChatGPT plugin which let's users interact with ChatGPT. This plugin is designed for dummies in-mind with no little coding requirements

Key Features

  • Plug and play: Simply import it in unity. Add your openAI key and start using ChatGPT inside unity
  • Chat History Toggle: Remember Chat context with a single click. With this toggle openAI API remembders user's last message and then it responds based on that message.

Technologies Used: Unity, C#, Openai API
